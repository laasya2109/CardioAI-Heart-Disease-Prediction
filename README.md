❤️ CardioAI – Heart Disease Prediction System

CardioAI is a web-based heart disease prediction system that uses machine learning (Random Forest algorithm) to assist doctors in identifying the risk of heart disease at an early stage. The application provides real-time predictions through a simple and user-friendly interface.

📌 Project Overview

Heart disease is one of the leading causes of death worldwide. Early detection plays a crucial role in prevention and treatment. CardioAI aims to support healthcare professionals by analyzing patient clinical parameters and predicting heart disease risk using a trained machine learning model.

✨ Features

🔐 Secure login for authorized users

🧠 Heart disease prediction using Random Forest

📋 Structured patient data input (13 clinical parameters)

⚡ Real-time prediction results

📊 Risk visualization with confidence score

🧾 Detailed assessment report with recommendations

🗂️ Patient records management (local storage)

🛠️ Technologies Used

Programming Language: Python

Machine Learning: Random Forest (Scikit-learn)

Web Framework: Flask

Frontend: HTML, CSS, JavaScript

Data Handling: Pandas, NumPy

Version Control: Git, GitHub
